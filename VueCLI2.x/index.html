# 安装 node.js

  • 下载node.js
  • 安装好后键入 node -v 以及 npm -v 来测试是否安装成功
  • 由于国内镜像下载很慢可以安装淘宝镜像,安装成功后，可以键入 cnpm -v 检查是否安装成功
npm install -g cnpm --registry=https://registry.npm.taobao.org

# 全局安装 vue-cli 脚手架

cnpm install -g vue-cli

# 全局安装 webpack 脚手架

cnpm install -g webpack

# 创建项目

vue init webpack 项目名称
cd 项目名称
cnpm i

# 运行项目

npm run dev
